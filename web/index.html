Api colores

El propósito de esta pequeña API es poder visualizar un color de un tamaño a elección.

El color puede elegirse en base al valor hexadecimal y se puede solicitar un formato json.

Endpoint: /api/:size

GET api/:size
api/100

Obtener un color (placeholder) de tamaño size.

GET api/:size?json=bool
api/100?json=true

Obtener un json del color (placeholder) de tamaño size.

GET api/:size?text=string
api/100?text=123

Obtener un color (placeholder) de tamaño size con un texto personalizado ("+" para que esté sin texto).

Endpoint: /api/:size/:hexColor

GET api/:size/:hexColor
api/100/805af3

Obtener un color de tamaño size de color hexColor.

GET api/:size/:hexColor?json=bool
api/100/805af3?json=true

Obtener un json del color de tamaño size.

GET api/:size?text=string
api/100?text=+

Obtener un color de tamaño size con un texto personalizado ("+" para que esté sin texto).
